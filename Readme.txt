The Web Series Pedia
This is an academic project in which we can find various information about web series.

How to run the The Web Series Pedia Project

1.Clone the Repository

2.Extract the file and copy mip folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4.Create a new user in sql server whose name=”fruit”, password= ”fruit123”.

5.Create a database with name WebSeries

6.Import WebSeries.sql file(given inside the zip package in SQL file folder)

7.Now type and run http://localhost/home.php on browser

8.You can create normal user or admin user account on create account page.
